## Installation

Follow these steps to run ChatboxAI locally:

1. **Clone the repository:**

   ```bash
   git clone https://github.com/your-username/ChatboxAI.git
   cd ChatboxAI
   ```

2. **Install dependencies:**

   ```bash
   npm install
   ```

   *(or `pip install -r requirements.txt` if using Python backend)*

3. **Run the application:**

   ```bash
   npm start
   ```

   *(or `python app.py` for Python backend)*

4. Open your browser at [http://localhost:3000](http://localhost:3000) (or your configured port).

---

## Usage

1. Sign in or create an account (if authentication is implemented).
2. Type your query in the chatbox and press **Enter**.
3. The AI will respond in real-time.
4. View conversation history, if stored.

---

## Configuration

* Add your API keys (if using OpenAI or another AI service) in `.env`:

```env
OPENAI_API_KEY=your_api_key_here
```

* Configure database credentials in `config.json` or `.env`.
